##【面试】重建二叉树

##
##一、描述

##
##　　输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树，假设输入的前序遍历和中序遍历的结果中都不含重复的数字，例如输入前序遍历序列{1,2,4,7,3,5,6,8	}和中序遍历序列{4,7,2,1,5,3,8,6	}，则重建出该二叉树。二叉树结点的定义如下	class BinaryTreeNode {    int m_nValue;    BinaryTreeNode m_pLeft;    BinaryTreeNode m_pRight;    public BinaryTreeNode(int m_nValue, BinaryTreeNode m_pLeft, BinaryTreeNode m_pRight) {        this.m_nValue = m_nValue;        this.m_pLeft = m_pLeft;        this.m_pRight = m_pRight;    	}	}

##
##二、解题思路

##
##　　可以根据前序遍历序列结合中序遍历序列找到某个结点的左子树和右子树，从而使用递归完成求解。
